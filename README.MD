# Quantum Option Pricing
### Classical vs Quantum Randomness in Monte Carlo Simulations

Monte Carlo option pricing using classical vs quantum random number generators, with traceable documentation and quality checks.

## 📂 Project Structure
- **README.md** — Overview and introduction
- **SPEC.md** — Technical specification
- **IMPLEMENTATION.docx** — Implementation details
- **BUSINESS REQUIREMENT SPECIFICATION.docx** — Full background, methodology, and traceability
- **/src** — Source code and test scripts

## 🚀 Features
- Classical Monte Carlo simulation for option pricing
- Quantum random number generator integration
- Histogram and convergence plot visualization
- Unit tests for reproducibility and quality assurance

## 📊 Deliverables
- Documentation (SPEC, IMPLEMENTATION, BUSINESS REQUIREMENT SPECIFICATION)
- Core modules:
  - `main.py` — Workflow orchestration
  - `monte_carlo.py` — Monte Carlo simulation
  - `quantum_rng.py` — Classical and quantum RNG
- Visualization scripts:
  - `visualization.py` — General plotting functions
  - `visualization_classical.py` — Classical convergence visualization
  - `visualization_quantum.py` — Quantum convergence visualization
- Test suite for reproducibility and quality assurance
